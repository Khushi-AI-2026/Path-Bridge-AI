## PROJECT OVERVIEW
PathBridge AI is an intellegent multi-agent AI system designed to bridge the gap between a student's current skills and target job requirements.It provides automated resume optimization , gap identification , and personalized learning paths to help students and reach thier carrer goals effectively.
